RxNorm

We are curating lists of drugs for common chronic diseases from RxNorm, which can be found below.

*Dates indicate RxNorm files queried in developing these datasets; any RxNorm CUI or NDC created after the end date will not be present, and similarly very old NDCs (no longer in use prior to 2005) are unlikely to be captured.
drug class id type description dates* updated
antihypertensives RXCUIs A SAS dataset containing a curated list of antihypertensive RXCUIs that includes only term types (TTY) allowable in the PCORnet common data model; also mapped are drug names and classes, both individually, and for combination products, a fixed-dose combination indicator, and the number of antihypertensive drugs associated with each RXCUI 2005 - 06Apr2020 11May2020
NDCs A SAS dataset containing a curated list of antihypertensive NDCs mapped from the above RXCUI file; as above, it includes only term types (TTY) allowable in the PCORnet common data model; drug names, classes, fixed-dose combination indicators and number of antihypertensive drugs are also mapped from the RXCUI file; note also only 11-digit NDCs are included as per PCORnet CDM specification 2005 - 06Apr2020 11May2020
